# MATTBEARCADE

MATTBEARCADE is the umbrella arcade ecosystem for MATTBEAR browser games, experiments, and playable neon machines.

## Shared Structure

```text
/engine
/shared-ui
/shared-audio
/shared-fx
/shared-fonts
/shared-save

```

## Philosophy

Build reusable systems once.
Reuse them everywhere.
Keep the arcade connected.
Avoid turning projects into disconnected haunted folders.
